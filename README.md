# Aura Intelligence — AI Assistant Framework

**Aura Intelligence** es un asistente virtual avanzado construido con **Angular 17+**, diseñado para ofrecer una experiencia de usuario premium mediante una interfaz minimalista y moderna con *glassmorphism*.

## ✨ Características
- **Interfaz Premium:** Diseño *Dark Mode* con estética Zinc/Violeta y tipografía Outfit.
- **Integración IA:** Conexión con modelos de lenguaje avanzados para tareas de ortografía, asistencia y más.
- **Reactividad:** Uso de Angular Signals para una gestión de estado eficiente y moderna.
- **Componentización:** Arquitectura modular escalable y fácil de mantener.

## 🛠️ Tecnologías
- **Angular 17+**
- **TypeScript**
- **Tailwind CSS**
- **OpenAI API**
- **Lucide Icons**

## 🚀 Instalación y Uso
1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Ejecutar el servidor de desarrollo:
   ```bash
   ng serve
   ```
3. Abrir en el navegador: `http://localhost:4200/`
